# Contribution type: review synthesis

Build the argument as:

`scope -> taxonomy -> comparison axes -> convergent evidence -> disagreements -> unresolved challenges -> research agenda`

- Define what is included, excluded, and meant by key terms.
- Organize literature by model level, regime, coupling strategy, evidence, or limitation rather than author chronology alone.
- Compare approaches on common axes such as assumptions, resolution, accuracy, cost, robustness, and interpretability.
- Separate field consensus, recurring limitations, and the reviewers' perspective.
- Make the outlook traceable to gaps established in the review.
