# Physics of Fluids leaning

Balance methodological clarity with a visible fluid-physics payoff.

- For method articles, use `identified defect -> corrected formulation -> focused tests -> physical consequence`.
- For application articles, keep the phenomenon and regime visible in the title and abstract.
- State whether the contribution is an Article, Letter, or review only when the source or target format specifies it.
- Avoid extending a short focused study into claims of general applicability.
