# Results

Structure each result unit as:

`question -> conditions -> observation -> quantitative support -> comparison -> immediate boundary`

- Use past tense for performed simulations or experiments and present tense for what a displayed figure shows.
- Name quantities, positions, averaging operations, times, and regimes.
- Report error, correlation, scaling exponent, balance, or uncertainty where available.
- Describe trends before mechanisms.
- Keep causal interpretation and broad implications for Discussion unless the journal combines the sections.
