# International Journal of Heat and Mass Transfer leaning

Connect the transport problem to a clear thermal or mass-transfer need, then define the configuration and governing parameters.

- Report heat- or mass-transfer quantities with normalization, boundary conditions, and property assumptions.
- Separate model verification/validation from parametric or application results.
- For performance studies, report workload, hardware, throughput, scaling definition, and parallel efficiency together.
- For multiscale work, state the scale hierarchy, information exchange, and regime coverage.
- Use Nusselt, Prandtl, Rayleigh, Peclet, optical thickness, temperature ratio, or related measures only after defining them.
