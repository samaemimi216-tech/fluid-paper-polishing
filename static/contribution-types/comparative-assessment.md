# Contribution type: comparative assessment

Build the argument as:

`shared problem -> precise algorithmic difference -> matched protocol -> accuracy/stability/cost results -> trade-off -> regime-specific recommendation`

- Hold equations, mesh, time horizon, boundary conditions, stopping criteria, error norm, and hardware fixed where relevant.
- Explain why the algorithmic difference should affect the measured outcome.
- Report disadvantages and cost penalties alongside advantages.
- Avoid declaring an overall winner when the evidence supports different choices in different regimes.
