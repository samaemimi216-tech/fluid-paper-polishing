# Theoretical study

Build the paper around `question -> assumptions -> formulation -> derivation -> limiting behavior -> implication`.

- State assumptions before using them and identify the parameter range in which they hold.
- Distinguish an exact result, asymptotic approximation, scaling argument, closure, and conjecture.
- Check limiting cases, dimensional consistency, conservation laws, and compatibility with known results.
- Explain what a derived relation predicts physically, not only how it is obtained.
