# Turbulence and data-driven conventions

- Define Reynolds number, averaging convention, wall units, domain, forcing, resolution, and statistical sampling.
- Distinguish DNS, LES, RANS, reduced-order, and data-driven closures.
- Report convergence of statistics, not only residual convergence.
- For learned models, state data origin, split strategy, inputs, outputs, baselines, metric, and tested Reynolds/geometry range.
- Separate predictive accuracy from physical interpretability and in-distribution performance from generalization.
- Require invariance, realizability, stability, and a-posteriori solver tests when such properties are claimed.
