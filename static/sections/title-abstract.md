# Title and abstract

Use a searchable title containing the phenomenon or method and the relevant regime or application. Avoid unsupported priority claims.

For an abstract, use:

`problem and regime -> limitation or gap -> proposed approach -> defining mechanism/property -> verification or validation -> main quantitative finding -> scope or implication`

- Define only indispensable abbreviations.
- Name the equations or physical model when central.
- Give quantitative evidence with conditions; do not list every test case.
- Distinguish properties built into the formulation from properties merely observed in tests.
- End with applicability and boundary, not generic future promise.
