# Corpus-calibrated move patterns

Select a pattern by contribution, then adapt it to the available evidence.

## Method abstract

`known method/context -> precise defect -> proposed correction -> operative mechanism -> guaranteed/recovered property -> focused tests -> quantified trade-off -> tested scope`

## Physical abstract

`phenomenon -> unresolved question -> setup/regime -> principal result -> mechanism or dominant parameter -> reduced relation/model -> implication`

## Data-driven abstract

`physical prediction/closure problem -> data and inputs -> model plus physical constraints -> held-out test -> generalization result -> interpretation -> limits`

## Review abstract

`why the area matters -> why synthesis is needed -> scope/taxonomy -> what is compared -> mature findings -> open challenges -> intended use`

## Introduction paragraph jobs

Map each paragraph to one dominant job:

1. physical/computational relevance;
2. model hierarchy or method families;
3. limitation organized by cause;
4. unresolved question;
5. present contribution and evidence plan.

Do not preserve a paragraph that mixes literature catalogue, derivation, contribution, and result preview merely because its sentences are grammatical.
