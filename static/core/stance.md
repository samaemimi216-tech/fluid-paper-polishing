# Core stance

- Put physics and evidence before elegance.
- Preserve the distinction between observation, numerical result, interpretation, and mechanism.
- Prefer precise conditions over broad praise: state the regime, geometry, parameter range, baseline, and metric.
- Use conventional fluid-mechanics language without copying source-paper phrasing.
- Keep Results and Discussion separable even when a journal combines them: report first, interpret second.
- Avoid promotional verbs and empty intensifiers such as `novel`, `remarkable`, `excellent`, `obviously`, and `significantly` unless novelty or statistical significance is demonstrated.
- Prefer `the present method` or the method name over repeated `we`; use active voice when it clarifies responsibility.
- Do not enforce a generic Nature cadence when technical density is necessary for reproducibility.
