# Experimental study

Build the paper around `physical question -> apparatus and conditions -> measurement -> uncertainty -> observation -> interpretation`.

- Report facility geometry, operating conditions, instrumentation, calibration, sampling, repeatability, and data reduction.
- Attach uncertainty to measured and derived quantities and distinguish random from systematic contributions when available.
- Use `measured`, `observed`, and `correlated with` for data; reserve causal verbs for controlled evidence.
- Distinguish measurement resolution from physical scale and detection limit from a zero value.
- Compare simulations and experiments using aligned definitions, locations, averaging windows, and boundary conditions.
