# Output format

Return:

1. `Polished text` as normal prose.
2. `Revision notes` with three to five concise bullets on logic, terminology, evidence, and journal fit.
3. `Author queries` only for missing information that affects correctness or reproducibility.

For a full section, add a one-line structural diagnosis. For a major rewrite, add a compact claim-evidence map with `supported`, `inferred`, or `needs evidence`. Show a terminology ledger only when the source contains drift or ambiguity. For side-by-side requests, use `Original | Revised | Reason`. Preserve citation keys, equation numbers, figure/table labels, and LaTeX commands exactly unless the user requests reformatting.
