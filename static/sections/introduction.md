# Introduction

Use a funnel adapted to fluid mechanics:

1. Establish the physical or engineering problem and controlling regimes.
2. Introduce the relevant model hierarchy and why common approaches struggle.
3. Group prior work by approach, assumption, or limitation.
4. State the unresolved gap precisely.
5. Present the contribution as a set of verifiable advances.
6. Briefly identify the evidence used to support them.

Avoid textbook histories, citation catalogues, application lists with no argumentative role, and detailed derivations. A numerical limitation should name its source, such as scale separation, stiffness, mesh constraint, numerical dissipation, conservation error, geometric complexity, or computational cost.
