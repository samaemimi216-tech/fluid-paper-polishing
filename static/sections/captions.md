# Figure and table captions

Make captions independently interpretable:

- identify the plotted or tabulated quantity, geometry, and case;
- define symbols, line styles, markers, normalization, and panels;
- report essential parameters and averaging or sampling conditions;
- identify reference data and error bars;
- avoid interpreting the result unless the journal permits discursive captions.

Use sentence case and consistent notation. Do not duplicate axis labels or the full Methods description.
