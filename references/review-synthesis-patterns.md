# Review synthesis patterns

## Build the taxonomy first

Choose an organizing axis that changes scientific interpretation:

- continuum, mesoscopic, kinetic, or molecular scale;
- domain-decomposition, hierarchical, embedded, or equation-free coupling;
- diffuse- versus sharp-interface treatment;
- pore-scale versus REV/upscaled description;
- supervised, unsupervised, physics-informed, or closure-learning approach.

## Synthesize each theme

Use:

`scope -> representative approaches -> common comparison axes -> consensus -> disagreement -> limiting cause -> research need`

Do not write one paragraph per paper. Cite representative evidence, then explain the pattern across studies.

## Outlook discipline

Each future direction must trace to one of:

- unresolved physical scale or regime;
- missing validation data;
- prohibitive cost or coupling difficulty;
- robustness, uncertainty, interpretability, or reproducibility gap;
- inconsistent terminology or benchmark practice.
