# Contribution type: physical investigation

Build the argument as:

`physical question -> controlling regime -> diagnostic or experiment -> observed structure/trend -> mechanism test -> reduced interpretation -> boundary`

- Lead with the phenomenon, not the software used to compute it.
- Identify the nondimensional groups that separate regimes.
- Report the observation before proposing a mechanism.
- Test rival explanations with budgets, correlations, conditional statistics, scaling, or controlled parameter changes when available.
- Distinguish a fitted correlation, reduced-order model, and general physical law.
